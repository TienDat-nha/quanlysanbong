TAI LIEU TOM TAT CAU TRUC DU AN SANBONG
Cap nhat: 2026-03-09

GHI CHU CAU TRUC MOI:
- Frontend da nam trong thu muc `frontend/`.
- Backend da nam trong thu muc `backend/`.
- Entry point backend hien tai la `backend/server.js`.
- Route backend hien tai tap trung trong `backend/routes/index.js`.
- Data layer MongoDB hien tai nam trong `backend/models/mongo.js`.

==================================================
1. MUC DICH TAI LIEU
==================================================
Tai lieu nay dung de:
- Giai thich nhanh cau truc source cho thanh vien moi.
- Biet file nao dung de lam gi, sua o dau.
- Hieu luong ket noi giua frontend, backend, MongoDB va email/payment.
- Hieu cac model/frontend view-model va model du lieu backend.


==================================================
2. TONG QUAN KIEN TRUC
==================================================
Du an dung:
- Frontend: React + react-router-dom.
- Backend: Express.
- Database: MongoDB.
- Email OTP: Nodemailer.
- Thanh toan dat coc: VNPAY, MoMo, chuyen khoan tinh.

Frontend duoc tach theo kieu gan MVC:
- View: nam trong src/views, chi render giao dien.
- Controller: nam trong src/controllers, noi view voi logic.
- Hook controller: cac file useXController.js chua state, side effects, xu ly su kien.
- Model: nam trong src/models, chua ham xu ly du lieu, validate, format, route map, API client.

Luong chinh:
src/index.js
-> src/App.js
-> src/controllers/AppController.jsx
-> Route den page controller
-> useXController.js lay / xu ly data
-> src/models/api.js goi backend
-> server/index.js xu ly request
-> server/mongo.js ket noi MongoDB
-> tra JSON ve frontend
-> Controller dua du lieu vao View de render.

Luong UI:
Controller -> View.

Luong du lieu:
Model API -> Backend API -> MongoDB.

3. SO DO KET NOI NHANH
==================================================
3.1. Boot app frontend
- src/index.js: tao React root va boc App bang BrowserRouter.

- src/controllers/AppController.jsx:
  - nap app.css.

- Page route trong AppController.
- Controller page (vi du FieldsController.jsx) goi hook logic.

- src/models/api.js dung fetch goi server/index.js.

  - doi cach xu ly loi.

- server/index.js khong tu noi truc tiep MongoClient.
- server/index.js goi ham trong server/mongo.js:
  - getUsersCollection()

  - getFieldsCollection()

- useAppController.js doc token trong localStorage.

- RegisterView -> RegisterController -> useRegisterController

- AdminFieldsView -> useAdminFieldsController
-> createAdminField + uploadAdminImage + getAdminDashboard

- DepositPaymentView -> useDepositPaymentController
-> getBookingDepositInfo / createVnpayDepositPayment / createMomoDepositPayment / confirmStaticDeposit
-> server/index.js
